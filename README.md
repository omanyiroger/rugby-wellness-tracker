Description
RCMIS is a web-based application designed to help rugby teams manage players, training, injuries, and physio care efficiently. It is built using *Flask, **SQLite, and **Tailwind CSS*, with role-based access for coaches, players, and physios.  

The system aims to streamline team management, track player availability, monitor injuries, and assist in planning training and match preparations.

---

 Features
 Coach Role
- Register and manage players
- Select first and second teams
- View player availability
- Track injury and physio status

Player Role
- Set availability for training days
- View team roster and assigned physio
- Specify strapping preference

physio Role
- View players needing strapping or medical attention
- Update player recovery and rehab status

Player Profile
- Name, position, date of birth, picture, height

---

Technologies Used
- *Backend:* Python, Flask
- *Database:* SQLite
- *Frontend:* Tailwind CSS
- *Authentication & Authorization:* Role-based access control
